# Claude Code Setup

## What It Is
Terminal-based coding assistant that runs directly in your command line

## Key Capabilities
* **File operations** - Search, read, edit files in your project
* **Terminal access** - Run commands directly from conversation
* **Web access** - Search docs, fetch code examples
* **MCP Server support** - Extend capabilities with specialized tools (databases, APIs, etc.)

## Platform Support
MacOS, Windows WSL, Linux

## Installation Steps
1. Install Node.js from nodejs.org/en/download
   - Check if installed: `npm help`
2. Install Claude Code: `npm install -g @anthropic-ai/claude-code`
3. Start and login: `claude`
   - First run prompts for Anthropic account login
